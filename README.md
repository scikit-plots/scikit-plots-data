scikit-plots-data
=================

Data repository for [scikit-plots](https://scikit-plots.github.io/dev/) examples.

| :warning: This is not a general-purpose data archive :warning: |
| :---: |

This repository exists only to provide a convenient target for the `scikitplot.load_dataset` function to download sample datasets from. Its existence makes it easy to document `scikit-plots` without confusing things by spending time loading and munging data. The datasets may change or be removed at any time if they are no longer useful for the `scikit-plots` documentation. Some of the datasets have also been modifed from their canonical sources.
